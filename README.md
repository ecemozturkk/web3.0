# Krypt - Web 3.0 Blockchain Application

By connecting your Metamask wallet with the application, it is possible to view the last transactions made, and send values through the Ethereum network on the blockchain. When sending the values, it is possible to write a keyword in the Keyword (GIF) field , through the Giphy API the keyword is associated with a similar gif, and then it is displayed on the transaction card. it is also possible to send a message with the description of the shipment.
